# OOP-Brain: C++ Sinir Ağı Kütüphanesi

Bu proje, nesne yönelimli programlama (OOP) prensipleri kullanılarak C++ dilinde geliştirilmiş modüler bir yapay sinir ağı kütüphanesidir. Geliştirme süreci, okuldaki 14 haftalık ders müfredatına paralel olarak aşama aşama ilerleyecek şekilde planlanmıştır.

## Ekip - Grup 7 (KOÜ | MCBÜ | HİTÜ)
Bu kütüphane, Kocaeli Üniversitesi, Manisa Celal Bayar Üniversitesi ve Hitit Üniversitesi öğrencilerinden oluşan 13 kişilik dev bir işbirliği ile geliştirilmektedir.

## Mimari Yapı
UML diyagramımıza sadık kalınarak sistem `process()` metotları üzerinden ileri iletimi sağlamakta ve çalışma zamanı polimorfizmi ile katmanları `NeuralNetwork` sınıfı altında entegre etmektedir.
